# @trustless/contracts

Foundry smart contracts for Trustless SocialFi platform.

## Prerequisites

- [Foundry](https://book.getfoundry.sh/getting-started/installation)

## Installation

```bash
forge install
```

## Build

```bash
pnpm build
```

## Test

```bash
pnpm test
```

## Lint

```bash
pnpm lint
```
